## [v26.1.7](https://github.com/redpanda-data/redpanda-operator/releases/tag/operator%2Fv26.1.7) - 2026-06-30
### Fixed
* Fixed `additionalCmdFlags` not being able to override the operator's default command-line flags in the rendered Deployment. Flags the chart sets a default for (e.g. `--enable-console`) were silently dropped in favor of the default; user-provided values now take precedence.
* Fixed `User` deletion getting stuck on clusters without an enterprise license. Schema Registry ACL cleanup no longer issues a `DeleteACLs` request when there are no ACLs to delete, which previously failed with "Invalid license: not present" and prevented the finalizer from being removed.
