## [v26.1.1](https://github.com/redpanda-data/redpanda-operator/releases/tag/charts%2Fredpanda%2Fv26.1.1) - 2026-03-31
### Added
* Added support for enabling TLS on external listeners independently of the internal listener's TLS state. External sub-listener TLS certificates are now correctly mounted even when the internal listener has TLS disabled.
* Added `allowPrivilegeEscalation: false` and `runAsNonRoot: true` to redpanda, redpanda-configurator, and sidecar container security contexts to address GKE security findings.
### Changed
* Bump chart version to install 26.1.1 releases by default
